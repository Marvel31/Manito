# To Do List

## 1. 행사 미참가자 처리

- 관리자 모드에서 참가자별 `참가 / 미참가` 상태를 설정할 수 있게 추가
- 미참가자는 참가자 로그인 목록에서 숨기거나 비활성화
- 미참가자는 마니또 배정, 제출 현황, 순위 선정, 슬라이드 쇼에서 제외
- 이미 등록된 사진, 예상, 한마디가 있을 경우 유지할지 초기화할지 정책 결정 필요

## 2. 마니또를 관리자가 미리 지정

- 관리자 모드에서 참가자별 `내가 찍어야 할 마니또`를 지정하는 기능 추가
- 참가자 화면에서는 본인의 마니또를 직접 입력하지 않도록 변경
- 참가자 오입력 방지
- 관리자 화면에서 마니또 배정 누락/중복 여부 확인 UI 추가 검토
- 본인이 본인을 마니또로 지정하지 못하도록 제한

## 3. 참가자 첫 입장 시 본인 확인 팝업

- 참가자가 이름을 선택하고 최초 입장할 때 확인 팝업 표시
- 예시 문구: `선택한 참가자가 "김민준" 본인이 맞나요?`
- 확인 후 비밀번호 설정 진행
- 취소 시 로그인 화면으로 복귀
- 참가자 이름 오선택 방지 목적

## 4. 마니또에게 한마디 작성 기능

- 참가자 사진 업로드 영역 아래에 `마니또에게 한마디` 입력란 추가
- 참가자는 사진과 함께 한마디를 저장 가능
- 여러 번 수정 가능하게 할지, 사진 재업로드 시 함께 갱신할지 정책 결정
- 관리자 제출 현황에 한마디 표시
- 슬라이드 쇼에서 사진, 미션, 참가자 이름과 함께 한마디 표시

## 5. 마니또 맞추기 성공 시 상대 미션 실패 처리

- 참가자가 `나를 뽑은 마니또`를 맞추면 해당 마니또의 미션을 실패 처리
- 이를 위해 관리자 배정 정보 기준으로 `누가 나를 찍는 사람인지` 계산 필요
- 관리자 화면에 미션 상태 표시
  - 성공
  - 실패
  - 미제출
  - 들킴
- 슬라이드 쇼 또는 제출 현황에서 실패 상태 표시
- 순위 선정 시 실패 처리된 제출물은 제외하거나 경고 표시

## 우선순위 추천

1. 마니또 관리자 사전 지정
2. 참가자 첫 입장 본인 확인 팝업
3. 미참가자 처리
4. 마니또에게 한마디
5. 맞추기 성공 시 상대 미션 실패 처리
